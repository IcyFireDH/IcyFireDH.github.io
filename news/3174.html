我们的青春，都有那么一位“金大侠”

发布时间：2018.11.01 19:14 阅读次数：142 出自：本站

郭襄回头过来，见张君宝头上伤口兀自汨汨流血，于是从怀中取出手帕，替他包扎。张君宝好生感激，欲待出言道谢，却见郭襄眼中泪光莹莹，心下大是奇怪，不知她为甚么伤心，道谢的言辞竟此便说不出口。

却听得杨过朗声说道：“今番良晤，豪兴不浅，他日江湖相逢，再当杯酒言欢。咱们就此别过。”说着袍袖一拂，携着小龙女之手，与神雕并肩下山。

其时明月在天，清风吹叶，树巅乌鸦呀啊而鸣，郭襄再也忍耐不住，泪珠夺眶而出。

白云聚了又散，散了又聚，

人生离合，亦复如斯。


“世间永别金庸，

无人再笑傲江湖”

1924-2018

我们的青春里，都有那么一位“金大侠”。25位南方电网人，想跟你聊聊与金庸有关的故事，以及许多不加掩饰的真实感悟。

//

“谢谢你给了我一个武侠梦”

//

“总是幻想自己变成武功高强的侠女”

@南方电网广东中山供电局谭茂茜


八九十年代，没有手机、没有电脑也没有网络，每个人都有一本《抄歌本》，而我的抄歌本，里面有很多金庸小说改编的港产电视剧里的词和曲。追星少女在紧张学习之余，一边偷摸看金庸的武侠小说，一边做了厚厚几本抄歌本，总是幻想自己变成武功高强的侠女。

梦中的江湖还在，而我没有成为“侠女”，江湖侠义之情、做人的正气品格，永远留在我心中。谢谢你，带给我“侠女梦”！

“有个别号‘令狐冲’，现在还在使用”

@南方电网广东汕头供电局安全监管部林湖生

外婆戴着眼镜看完整部《射雕英雄传》，间中和我偶尔还谈谈她的看法。外婆在我心中就如“黄蓉”般娇美、俏皮；曾经认为我如果在过去一定是位“侠女”，现在也改不了爱打抱不平。

从事的职业最长的经历是“电网卫士”，现在是全职“电网保镖”，这是一种怎样的心路历程？还有个别号“令狐冲”，现在还在使用。

“再平凡的人也可以成为有用之才”

@南方电网海南三沙供电局周丹萍

金庸先生在我们手里这小小的一方天地中，为我们描绘出了江湖梦：无论是义薄云天的乔峰、侠之大者为国为民的郭靖，还是侠骨柔情的段誉与狂傲不羁的杨过，这些书中的偶像，都以润物细无声的方式影响着我们的成长。

纵然不能仗剑走天涯，仍怀侠义于心中……追求理想，再平凡的人也可以成为有用之才！

“侠之大者，为国为民”

@南方电网广州增城供电局杨景鸿

金庸的作品描写的大都是主人公行走江湖，历经磨难，最后成长为盖世大侠的故事。他的作品让我从小就受到善良、勇敢、正直和坚强的熏陶，让我懂得了乱世中的家国情怀。

“侠之大者，为国为民。”或许今天的社会已经不需要武功出众的侠客了，但是书中所展现的为国为民的大侠精神，仍是需要我们去弘扬。

“我的作文总是有情节有故事有温度”

@南方电网广西玉林北流供电局李木茂

惊闻金庸先生去世，急从书房里找出《射雕英雄传》重读。三十年前，这套书我看过，我妈妈看过，我大姑妈也看过。她们是农村妇女，但也忙里偷闲在昏黄的灯光下看金庸的小说，而我也是窝在被窝里偷偷看的。

郭靖智力平平，却能一步步走向成功，杨康资质那么好到头来却一败涂地，这对我影响巨大！事实证明我偷读小说，也没影响学习，相反我的字写得很潇洒，我的作文总是有情节有故事有温度，一直都得到老师的表扬。

“金庸的小说成就了我心中的大侠梦”

@南方电网海南海口供电局 杨永光

对于90后的我来说，金庸的小说是我童年最美好的回忆。特别喜欢乔峰的降龙十八掌，每当他使出降龙十八掌攻打坏人时，我都会不由主地模仿，甚至有时候跟小朋友打闹时就像乔峰附体一样，打出一套降龙十八掌。金庸让我的童年多了美好的回忆，也成就了我心中的大侠梦。

“金庸走了，阿朱就是真的死了吧”

@南方电网深圳龙岗供电局曾钰

爷辈父辈还有我这辈三代人，就算没看过金庸的小说也看过相关影视节目，小时候打架处于下风时一套降龙十八掌喊出来也是如有神助。金庸让我们知道了什么叫“侠之大者，为国为民”，知道了什么叫“无招胜有招”，知道了什么叫“我就喜欢阿紫，她就不能瞎”。既然金庸走了，那么，阿朱就是真的死了吧。

“心中还有一把不曾熄灭的火”

@南方电网云南红河泸西供电局吴勇

弱冠时，幻想能有杨过、段誉、张无忌的洞中奇遇，更憧憬长大像独孤求败一样，纵横江湖三十余载。

成年时，面对成长中的挫折与失意，更艳羡令狐冲的放荡不羁、执酒当歌、快意江湖。

而今四十不惑，肩上担子与责任更沉，再次翻起发黄的小说，“侠之大者，为国为民”，郭靖与萧峰的家国情怀又让我经历了一次庄严的洗礼，疲惫之感荡然无存。

左手是岁月的残酒，右手是钝去的剑锋，我们终究没能把它握在手里，但心中一把不曾熄灭的火，依然执着地燃烧着。

//

“在奇幻的世界里我遇见了真实”

//

“查老一去，世上再无“江湖”？我说非也”

@南方电网广东珠海斗门供电局 陈建文

查老用看似天下任我闯的快意江湖小说，告诉我们做人要忠孝义存的道理。

有的人说，查老一去，世上再无“江湖”，我说非也，消失的只是上一个时代的“江湖”。江山代有才人出，下一个“江湖”要靠一代又一代的人为祖国的强大奋斗终身，用默默耕耘成就一代又一代的工匠宗师，将我们的技术传遍五湖四海，谓之“江湖”也。

“或许金庸还在笑着说，你们难过什么？‘侠’还在呢”

@南方电网广州供电局输电管理二所钱泓廷


（当年，我们还拍摄了一部《输电大侠养成记》的微电影。电力江湖也是江湖，里面有我们的喜怒哀乐，也有着我们的汗水和奋斗。）

什么是大侠？金庸先生告诉我，人生的路诱惑太多，能不忘初心的人，是大侠；能心胸开阔的人，是大侠；能把学识真真吸收随时可用、而不是花拳绣腿的人，是大侠；能一心一意、专注无争的人，是大侠。

所以，江湖没有离开，时代也还在继续，因为“侠”的精神永存。

“所谓精修，御于物而不惑于物”

@南方电网国际公司老挝南塔河公司杨毅

金庸在《神雕侠侣》中借独孤求败之口说：“凌厉刚猛，无坚不摧，弱冠前以之与河朔群雄争锋。紫薇软剑，三十岁以前所用。重剑无锋，大巧不工，四十岁以前持之横行天下。四十岁后，不滞于物，草木竹石均可为剑。自此精修，渐进于无剑胜有剑之境。”

所谓精修，修己修道而不愿与人较之高下长短也。不愿与人相较而勤于自修者，则御于物而不惑于物，于是能渐进于无，胜于有。共勉！

“在自己的岗位上，做自己的大侠”

@南方电网广东佛山顺德供电局容桂供电所李国霖

学生时代的我时常怀揣着一颗武侠心，那段时间，班主任多次捉到我们在上课偷看小说，还对我们说，小说里的大侠被人敬佩，是因为他们能够知道什么时候该做什么。

对啊，真正的大侠不会沉浸自己的梦中。在生活中做一个敢爱敢恨的人，在工作中踏实勤奋、不骄不躁。虽然我只是一个基层电力工人，但我相信，只要兢兢业业奋斗在自己的岗位上，就能做自己的大侠。

“永远心怀赤诚”

@南方电网广西崇左供电局市场营销部梁婷婷


（当年还装扮起黄药师的关门徒弟程英。）

告别金庸先生，告别的是我所有美好的青春回忆。金庸先生的作品对我性格影响很大：工作中雷厉风行，干练大方，虽然做不到“侠之大者，为国为民”，但也是心怀赤诚、认真工作、服务南疆的“女侠”一名吧。

“只要用心”

@南方电网广东阳江供电局张珀瑜

“大丈夫一生当为国为民，方为真正的侠之大者”。小时候总渴望穿着长衣与“侠”为伴。而如今，我穿上了蓝色工装，尽绵薄之力不足为奇，依然愿如郭靖守襄阳时般，说声：“对，用心服务”。感谢老先生给予的源源不断的力量。

“不再紧盯成败、计较得失”

@南方电网广东肇庆高要供电局白国颖

印象最深的是那藏书阁的扫地僧。一招一式一席话，化解了萧、慕容两家多年的恩怨情仇，使他们摒弃那迷惑心智的戾气，大彻大悟，自我升华。

扫地僧的处世方式，让我在工作中更加注重对自身能力的锻炼与提升，不再紧盯成败、计较得失，而是沉下身子去做好每一件事，使自己一次次蜕变，得到一点点的提高。只有扎扎实实做好每一件事，才可能有一天“当惊世界殊”。

“江湖莫谈儿女情，江湖儿女情更深”

@南方电网云南曲靖师宗供电局盛亚


书看得多了，侠气入骨，小子乐于助人，坚信正义，还时不时做着路见不平一声吼的“傻事”。现在，我是一名电网人，说起保电送温暖，还颇有些现代大侠的意味。嗯，女儿身，大侠心，身在电网，心系万家，总是应了那句“江湖莫谈儿女情，江湖儿女情更深”。

“在读书之外，还有多种多样的学习方式”

@南方电网广东珠海斗门供电局初润田

郭靖勤学苦练十余载，但进展奇慢。马钰一针见血：“这是教而不明其法，学而不得其道。”正是因为江南七怪“教而不明其法”，郭靖才会“学而不得其道”。从郭靖的武学成长经历，我明白学习并不是一件简单的事情，光凭一身蛮力并不一定能有多大效果。

《神雕侠侣》中，我最喜欢的是程英，知进退，懂取舍，世事洞察皆学问、人情练达即文章。韦小宝的故事则告诉我们，读书只是学习的一种方式，在读书之外，还有多种多样的学习方式，工作中亦是如此。

“这不正是我的写照吗？”

@南方电网深圳宝安供电局李瑞

年少轻狂的岁月里，都是那个炙热的武侠梦伴随我。从少年时期走到而立之年，随着他哭，跟着笑，由懵懂到心跳……正如郭靖，年轻时不管天高地厚，人到而立之年，又渐渐收敛锋芒，变得谦卑从容。这不正是我的写照吗？

“偷偷看金庸的青春，不会再有了”

//

“我的青春只有那么一位‘金大侠’”

@南方电网云南昆明供电局变电修试所周帆

金庸说，自己前世大概是大理人。六脉神剑，一阳指，无量山，崇圣寺，苍山，洱海……大理在金庸老先生的笔下，增添了几分神秘色彩。身为大理人，也倍感骄傲。

我们的少年时代没有手机，也没有电脑，有的是金庸先生的武侠江湖。当年胡军版《天龙八部》在大理开拍的时候，在我们当地引发了轰动，小伙伴们成群结队地前去大理，只为看一眼“武侠”，江湖。

我的青春只有那么一位“金大侠”。

“这大抵是我这辈子唯一一次‘偷东西’”

@南方电网贵州兴义供电局龙玉玺

小学三年级开始看武侠，在课桌上凿个洞，一边假装认真听课一边手在桌箱里不停地翻书。彼时有个同学经常拿他哥哥的小说在教室里炫耀，而我则争做值日，晚上把他的书悄悄顺回家，第二天一大早还回来，我就是这么看完《连城诀》和《鹿鼎记》的，这大抵是我这辈子唯一一次“偷东西”。

后来，我刻意模仿金老爷子飘逸厚重的文风，写了篇稿件叫《侠骨丹心闯锅圈》，才知道，金老爷子一直被模仿，从未被超越。

“我给乔峰和阿朱写了一个结局”

@南方电网广东肇庆供电局李天毅

初中读到阿朱在乔峰怀里闭眼，我哭得稀里哗啦。乔峰决绝一刻，我又大哭了一场。心中不忍结局，自己改写了一段，让乔峰和阿朱重逢，归隐草原，放马牧羊。后来，被老师发现了我上课偷看小说和写小说，于是没收了我的书和小笔记。可是我当时一点儿也不后悔。

忽然，我很想知道，当年没收我书的那些老师，是不是也在怀念老先生？

“他很大程度塑造了我的三观”

@南方电网广州供电局物流中心李虎声


（十年前，突发奇想在QQ空间写下文章妄评金庸笔下的十大高手。）

在我的青葱岁月中，几乎每天都有金大侠的“江湖”，他很大程度塑造了我的人生观、价值观和爱情观。在我的眼中，金庸有萧峰的豪气干云，有郭靖的侠之大也，有段誉的谦逊有礼，有令狐冲的潇洒飘逸，有张无忌的与世无争，更有韦小宝的多情风流。

“课堂上偷看‘抢借’来的小说”

@南方电网广东茂名供电局项目管理中心杨芒生

在课堂上总也想方设法、自欺欺人偷看“抢借”来的小说。那时候总以为金庸是古代的大侠，老先生笔下《倚天屠龙记》中的赵敏、《天龙八部》中的阿朱……很多角色都让人印象深刻。有时候幻想里面的英雄大侠就是自己，胆也大了，气也神了，走路都有飞沙走石的感觉了……

“我竟然买了一本《如来神掌》秘籍……”

@南方电网海南琼海供电局宋维腊


记得第一次看武侠小说是在小学四年级，便是金庸的《连城诀》，从此一发不可收拾。有次在外仗剑走天涯，发现路边小店有卖《如来神掌》武功秘籍，当时如获至宝！后来发现原来是笔记本，但还是买下来以留纪念。

“半夜大哭不敢出声怕被爹妈发现”

@南方电网海南儋州供电局洋浦供电公司尹舵

有一次看《飞狐外传》程灵素死的时候，哭的那叫一个惨，偏偏还是半夜偷偷看书，不敢出声怕被爹妈发现，忍不住熬夜给老爷子写了几千字的信，最后投寄无门给留了下来。

飞雪连天射白鹿，笑书神侠倚碧鸳，可不是张无忌在光明顶大杀四方，萧峰三兄弟在少林寺战群雄才好看，每个小人物、每个配角我恨不得都在心里安排个自传。


●唐微刚摄

冰雪来袭，勇敢攀登。


●余云华摄

身后就是悬崖，从不退却。


●刘晖摄

眼前就是深渊，勇往直前。


●姚其欣摄

保家卫国，无怨无悔。


●李品摄

挑战“世界第一”，不断突破。


●金建冲摄

常怀赤子心，胸中有火，眼里有光。

侠之大者，

为国为民。

是你让我懂得了这个道理。

谢谢你，金庸。

再见，金庸。
